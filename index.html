Tutorial: Como configurar a API Google Gemini

1 Acesse o Google AI Studio
Vá para https://aistudio.google.com/app/apikey e faça login com sua conta Google.
2 Crie uma nova chave de API
Clique no botão "Criar chave de API" e selecione o projeto que deseja usar ou crie um novo.
3 Copie a chave gerada
Após a criação, copie a chave de API exibida. Importante: Esta chave não será mostrada novamente, então salve-a em um local seguro.
4 Cole a chave no campo acima
Volte para esta página e cole a chave no campo "Chave da API Google Gemini".
5 Teste a conexão
Clique em "Testar API" para verificar se a chave está funcionando corretamente.
